Overview and Features

How to put the template online

After you have downloaded the theme to your liking by replacing the text and images with your own, you'll want to put it online but in order to be able to put your new theme online there are some requirements you need to meet:

  1. Buy a domain name
  2. Buy a web hosting service
  3. Have a ftp client installed

SASS

Using SASS you have more control over which components to include.  You can find them in folder assets/sass:

  • _document.scss
  • _elements.scss
  • _layout.scss
  • _pages.scss
  • _posts.scss
  • _reset.scss
  • _support.scss
  • _user_dashboard.scss
  • _variables.scss
  • _widgets.scss
  • style.scss

We recommented you use koala-app  to compile SASS files.

 

Template color

You can change color in file assets/sass/_variable.scss

// Default Color:
$text:           #666666;
//$text:         #8d9aa5;

$primary:     #00979d;

$hover:          #ff9900;

$secondary:    #ff9900;

$border:      #e9e9e9;

$meta:       #AAAAAA;

$primarybg:       #222222; //darkslateblue
$primary_text:  rgba(255, 255, 255, 0.6);

$code:           #b9dc2f;
$sale:           #ea4c89;
$print:       #2d3538;

Javascripts

Base Structure

This theme contains many advanced jQuery functions and libraties.

All javascript files (at assets/js/ folder) :

  • jquery-1.11.1.min.js
  • config.js
  • libs.js
  • libs/retinal.min.js
  • libs/owl.carousel.js
  • libs/semantic.min.js
  • libs/modernizr.min.js
  • libs/fitVids.min.js
  • global.js

If you feel the need to customize any functions for any reason, open and edit custom.js file in your theme's scripts directory assets/js/
